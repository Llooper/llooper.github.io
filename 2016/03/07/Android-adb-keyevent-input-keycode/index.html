Android adb input keyevent "keycode"

  • Android

Andorid adb input keyevent “keycode”整理，方便terminal输入Androidkeyevent事件。

adb 输入keyevent事件例子：

adb input keyevent 66
adb input keyevent KEYCODE_ENTER
主要有以上俩种形式，当然，数字比较多记起来繁琐，整理网络上的一些汇总成为此篇page。
